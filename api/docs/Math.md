## 描述
**

Math是一个对象，定义了引用有的算术函数和常量的属性。使用如下的语法就可以调用它们：  
    y = Math.sin(x);  
    area = radius \* radius \* Math.PI;  
  
  
Math并不像Date和String那样是对象类。没有Math()构造函数，像Math.sin()这样的函数只是函数，不是对象的方法。
