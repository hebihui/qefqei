# Date.toString() _ECMAScript v1_

将Date转换为字符串

## 摘要

date.toString()

## 返回值

date的字符串表示，是人们可以读懂的，用本地时间表示。

## 描述

方法toString()返回一个人们可以读懂的、由实现决定的日期的字符串表示。它与toUTCString()的不同，toString()以本地时间表示日期。而它与toLocaleString()的不同之处在于，它不使用地方规则采用的形式表示日期和时间。

Date.parse()  
Date.toDateString()  
Date.toLocaleString()
