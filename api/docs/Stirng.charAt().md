# Stirng.charAt() _ECMAScript v1_

返回字符串中的第n个字符

## 摘要

stirng.charAt(n)

## 参数

n  
    应该返回的字符在string中的下标。

## 返回值

字符串string的第n个字符。

## 描述

方法Stirng.charAt()返回字符串string中的第n个字符。字符串中第一个字符的下标值是0。如果参数n不在0和string.length-1之间，该方法将返回一个空字符串。注意，JavaScript并没有一种有异于字符串类型的字符数据类型，所以返回的字符是长度为1的字符串。

## 参阅

String.charCodeAt()  
String.indexOf()  
String.lastIndexOf()
